Gym Management System - README


-------Introduction-------

The Gym Management System is a web-based application designed to streamline gym operations such as member registration, attendance tracking, payment management, and staff coordination. This system is built using PHP, JavaScript, HTML, and CSS, with MySQL as the database.




-------System Requirements-------

To successfully run the Gym Management System, ensure that your system meets the following requirements:

Software Requirements:

XAMPP (Apache, MySQL, PHP, Perl)

VS Code (or any other code editor/IDE)

Web Browser (Chrome, Firefox, Edge, etc.)


Installation Instructions:

Download and Install XAMPP:

Download XAMPP from Apache Friends

Install and start the Apache and MySQL services from the XAMPP control panel.


Setup the Database:

Open phpMyAdmin (http://localhost/phpmyadmin/)

Create a new database: brew+flex

Import the provided brew+flex.sql file to set up the necessary tables and data.


Run the Application:

Start Apache and MySQL from the XAMPP control panel.

Open a web browser and navigate to http://localhost/brew+flex





----------------Troubleshooting----------------

Database Connection Error: Ensure MySQL is running and credentials in config.php are correct.

404 Page Not Found: Make sure the project folder is inside htdocs and the correct URL is used.

Login Issues: Check user credentials and ensure they exist in the database.

For any issues, refer to XAMPP logs or contact support.
